Para ayudarte a visualizar la nueva estructura del panel, he creado un croquis simple. Este diseño sigue las peticiones de tener etiquetas de selección bien posicionadas y el contenido organizado.

Aquí tienes un esquema de cómo se verían los elementos principales en el panel:

```
+---------------------------------------------------------------------------------+
|                                                                                 |
|                                [MY CHEAT SHEET]                          [X]    |
|                                                                                 |
|  +---------------------+   +-----------------------+   +---------------------+  |
|  |       CLASE         |   |    ESPECIALIZACIÓN    |   |      CONTENIDO      |  |
|  +---------------------+   +-----------------------+   +---------------------+  |
|  | [ Dropdown Clase ]  |   | [ Dropdown Espec. ]   |   | [ Dropdown Cont. ]  |  |
|  +---------------------+   +-----------------------+   +---------------------+  |
|                                                                                 |
|  +---------------------------------------------------------------------------+  |
|  |                      [ SCROLLABLE CONTENT FRAME ]                         |  |
|  |                                                                           |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |  |  PRIORIDAD DE ESTADÍSTICAS                                          |  |  |
|  |  |    Intelecto > Celeridad = Maestría > Crítico = Versatilidad        |  |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |                                                                           |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |  |  ARMAS - BEST IN SLOT                                               |  |  |
|  |  |    [icono] [icono] [icono]                                          |  |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |                                                                           |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |  |  ARMAS - ALTERNATIVAS                                               |  |  |
|  |  |    [icono] [icono] [icono]                                          |  |  |
|  |  +---------------------------------------------------------------------+  |  |
|  |                                                                           |  |
|  |                       ... (Más secciones de equipo) ...                   |  |
|  |                                                                           |  |
|  |                                                                           |  |
|  +---------------------------------------------------------------------------+  |
|                                                                                 |
|  [ Importar/Exportar ]    [ Editar ]                                            |
|                                                                                 |
+---------------------------------------------------------------------------------+
```

Este esquema ilustra:

  * **Una fila superior para las opciones de filtro:** Las etiquetas y sus respectivos menús desplegables están horizontalmente alineados, facilitando la selección de la clase, especialización y tipo de contenido.
  * **Un área de contenido principal y deslizable:** El `ScrollFrame` ocupa la mayor parte del panel. Dentro de este marco, las secciones como "Prioridad de estadísticas" y los elementos de equipo están organizados verticalmente, cada una con su propio recuadro para una separación visual clara.
  * **Botones de acción:** Los botones "Importar/Exportar" y "Editar" se mantienen en la parte inferior del panel, bien definidos y separados del contenido principal.
